# Pterodactyl Installation Scripts

Easy to install pterodactyl or add small additions like phpMyAdmin or a database host then read on

## Features

- Almost everything can be created for you automatically
- Your node and database host are directly linked to your panel
- Possibility to install phpMyAdmin

## How do I use the script?
To use this script you need to run it as root user

```bash
wget https://pterodactyl.thomas07.eu/script.bash
bash script.bash
```
Once you have done that you can choose which installation you want to run if you want to do things like phpMyAdmin or create a Database Host then you have to select Other Options
